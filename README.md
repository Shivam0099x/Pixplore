# 🔍 Pixplore

**Pixplore** is a modern media search application built with **React**, **Redux Toolkit**, and **Tailwind CSS**. It allows users to search for **photos**, **videos**, and **GIFs** from multiple media providers through a clean and responsive interface.

Users can also save their favorite media locally using **localStorage**, making them available even after refreshing the application.

---

## ✨ Features

- 🔍 Search media by keyword
- 📷 Browse high-quality photos
- 🎥 Search videos
- 😂 Explore GIFs
- ❤️ Save favorite media locally
- 📱 Fully responsive design
- 🔄 Tab-based media navigation
- 📦 Global state management with Redux Toolkit
- 💾 Persistent favorites using localStorage

---

## 🛠️ Tech Stack

### Frontend

- React.js
- Tailwind CSS
- Redux Toolkit
- React Router DOM
- Axios

### APIs

- Unsplash API (Photos)
- Youtube Video API
- Giphy GIF API
- Giphy Stickers API


## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Shivam0099x/Pixplore.git
```

Navigate into the project

```bash
cd Pixplore
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_UNSPLASH_API_KEY=your_key
VITE_YOUTUBE_API_KEY=your_key
VITE_GIPHY_API_KEY=your_key
```

Start the development server

```bash
npm run dev
```
